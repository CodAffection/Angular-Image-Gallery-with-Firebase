# Angular Image Gallery with Firebase Cloud Storage

How to upload an image to firebase cloud storage in angular 7 and how to retrieve firebase file back to the angular project.

## Get the Code

```
$ git clone https://github.com/CodAffection/Angular-Image-Gallery-with-Firebase.git
$ cd Angular-Image-Gallery-with-Firebase/Angular 7
$ npm install
```
